# 🚀 Aureon AI Lead Management & CRM Assistant

An AI-powered Lead Management & CRM Automation System built with **n8n**, **Google Gemini**, **Pinecone**, **Google Sheets**, and **Gmail**.

The system automatically qualifies leads, extracts business information, recommends the best AI solution, stores the lead inside a CRM, and sends a personalized confirmation email—all without manual intervention.

---

# 📌 Project Overview

This project demonstrates how AI can automate the entire lead management process.

Instead of manually collecting customer information, updating spreadsheets, and writing emails, the AI Assistant performs the complete workflow automatically.

The assistant:

- Collects customer information
- Qualifies leads intelligently
- Recommends the best AI solution
- Saves lead information into Google Sheets CRM
- Sends personalized Gmail confirmation emails
- Maintains conversation context using AI Memory
- Retrieves company knowledge using Pinecone Vector Database
